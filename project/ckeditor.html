GNU v2许可

修复细节
  • #5125：固定：  通过键盘 键删除禁用了 autoParagraphbackspace的小部件 会删除编辑器可编辑区域并使编辑器崩溃。
  • #5135：固定：如 文档中所述，checkbox.setValue 和 radio.setValue 方法不可链接。感谢 乔丹布拉德福德！
  • #5085：固定： 语言 插件删除标记外语文本的元素，如果该元素没有关于文本方向的信息。
  • #4284：固定： 表选择 合并具有行跨度的单元格会引发意外错误，并且不会创建撤消步骤。
  • #5184：固定： 编辑器占位符 插件会降低打字性能。
  • #5158：固定： CKEDITOR.tools#convertToPx() 如果从 DOM 中删除了辅助计算器元素，则会给出无效结果。
  • #5234：固定：  Easy Image 不允许使用工具栏按钮上传图像文件。
  • #438 ：固定：无法通过键盘 从 工具栏导航到 元素 路径，反之亦然。
  • #4449：固定： dialog.validate#functions 错误地组合返回可选错误消息的函数，例如 dialog.validate.number 由于不必要的返回类型强制。
  • #4473：固定：  dialog.validate 方法不接受参数值。该问题起源于 dialog.validate.functions 方法，该方法没有正确地将参数值传播到验证器。受影响的验证者：
    functions
    equals
    notEqual
    cssLength
    htmlLength
    inlineStyle
  • #5147：固定：“ 辅助功能帮助” 对话框不包含有关在离开对话框时将焦点移回编辑区域的信息。
  • #5144：固定： 菜单按钮 和 面板按钮 在浏览器的辅助功能树中错误地指示其关联弹出菜单的打开状态。
  • #5022：固定： 查找和替换 不响应 Enter 键。
API 变更：
  • #5184：添加了 config.editorplaceholder_delay 配置选项，允许在更改编辑器内容时在切换占位符之前延迟占位符。
  • #5184：添加了 CKEDITOR.tools#debounce() 允许将传递的函数执行推迟到自上次调用以来经过给定毫秒数的函数。
